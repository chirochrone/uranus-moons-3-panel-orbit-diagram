This Python script is intended to be executed via command-line.

This script plots the 3D orbits of Uranus's major and irregular moons by querying JPL Horizons (astroquery) for their positions over time, and then plotting them in 3D space.

Moon orbit data obtained from JPL (https://ssd.jpl.nasa.gov/sats/elem/sep.html), code partially written by Claude AI, fact-checked and corrected by Nrco0e.
